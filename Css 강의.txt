문서를 꾸미는 역할
내부 <style></style>
외부 <link>
    -링크 속성 <link rel="stylesheet' type="text/css" href="외부 CSS 경로">

정의 방법

적용 대상 {
    스타일 정의
}

적용 취소, 적용에 대한 설명 - 주석
주석 다는 방법 : /* (시작) ~ */ (끝)

선택자
    - 태그 선택자 (태그명으로 선택)
    
    - 클래스 선택자
        .클래스명(마침표)

    - 아이디 선택자
        #아이디명(샵)

    - 태그의 style 속성으로 정의(선택)하는 방법
    
    * 적용 범위
     태그 > 클래스 > 아이디 > 태그의 스타일 속성

CSS 적용 우선 순위
 (적용 범위가 좁을 수록 우선 순위가 '높다'.)

태그의 style 속성 > 아이디 > 클래스 > 태그

 같은 우선순의 등급의 클래스 선택자의 경우 
 나중에 입력한 클래스 선택자가 우선적용된다.

 우선순위를 코더가 강제로 조정적용
 !important -> 우선순위 강제 적용

조상 자손 선택자
왼쪽 -> 오른쪽 (일반적으로 왼쪽으로 갈수록 상위 태그,
                오른쪽으로 갈수록 하위 태그)

부모 자식 선택자
    바로 위 층위: 부모선택자
    바로 아래 층위: 자식 선택자
        부모 > 자식

그룹 선택자
    콤마(,)로 구분하여 여러 선택자를 적용

가상 클래스 선택자
    - 선택자: 상태값
    
    요소의 상태로 선택

    ex) 체크, 라디오 박스 : 체크가 되어있는지?
        선택자:checked
    ex2) 요소에 마우스가 올려 있는 상태인지??
        선택자:hover
    ex3) 요소가 읽기 전용 상태인가?
        선택자:read-only
    ex4) 선택자가 아닌 상태인가?
        선택자1에 선택자2가 포함이 되어 있지 않은 상태인가?
        선택자1:not("선택자2")

    EX) 첫번째 자식인가?
        선택자:first-child
    EX2) 마지막 자식인가?
        선택자:last-child
    Ex3) 몇번째 자식인가?
        선택자:nth-child
        nth-child(숫자) -> n번째 요소
        nth-child(수자) -> 수식 계산 후 선택
        
        2n -> 2,4,6,8,10 ... (n은 1부터 시작)
        2n+1 ->  1,3,5,7,9 ..... (n은 0부터 시작)

    - 가상으로 추가하여 선택하는 선택자
        선택자: after {content:''}
            * 선택자의 자손으로 가장 끝에 추가하여
    
          선택자:before
            * 선택자의 자손으로 가장 앞에 추가


==== 결합자 ====

복수의 선택자를, 특정 조건을 걸어서 선택한뒤, 
한번에 스타일을 적용시킴.(좀더 짧게 코딩할수 있음)

부모 > 자식

형제 

인접 형제(+)
선택자1 + 선택자2 -> 선택자1(왼쪽 위치)을 인접형제로 가진 선택자2를 선택

일반 형제(~)
선택자1~선택자2 -> 선택자1(왼쪽 위치)을 형제로 가지고 있는 선택자2를 선택

요소 속성 선택자
선택자 [속성명='속성값'] {}
