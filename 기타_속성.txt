text-decoration
    - non - 기본값
    - underline - 밑줄
    - italic - 이탤릭체(기울임체)
    

list-style
    ul 태그의 li 스타일
        disk, circle, square

        none-> 스타일 없음

cursor
    - 마우스를 올렸을 때 화살표 모양
    pointer -> 손가락 모양
    hand

border: 1px solid red; //모든 방향
    -border-top : 상단 경계선
    -border-bottom  : 하단 경계선
    -border-left : 왼쪽 경계선
    -border-right : 오른쪽 경계선

height - 높이
    min-height : 최소 높이
    max-height : 최대 높이

width - 너비
    min-width : 최소 너비
    max-width : 최대 너비

margin
    -auto (양 방향 균등 여백)
    -margin:0 auto (좌우로 균등 여백-가운데 정렬)

text-transform
    :uppercase (소문자 -> 대문자로)
    :lowercase (대문자 -> 소문자로)

vertical-align: 수직 정렬
    -top
    -middle
    -bottom

text-decoration:
    - none: 효과 없음
    - underline : 밑줄

text-indent
    - 들여쓰기

    예) text-indent: 10px;
    -> 좌에서 우로 10px만큼 들여쓰기.

text-overflow: 텍스트가 넘쳐흐를때
    - ellipsis (생략 표시 ...을 추가)

+ 생략되는 글 요소가 너비를 가져야 하고 
+ 글 개행 속성이 없어야 한다.

white-space
    -nowrap -> 줄개행x
    -wrap -> 줄 개행
 + overflow: hidden -> 넘치는 부분 가리기
